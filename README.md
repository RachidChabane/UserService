# User Service

- Copier le .env présent sur le discord dans le channel lien à la racine du projet.

- Installer les dépendances :  
```npm install```

- Lancer la base de données :  
```npm run docker:dev-db:start```

- Générer les models prisma
```npm run db:push```

- Lancer le serveur :  
```npm run dev```
